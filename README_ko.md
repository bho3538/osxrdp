# osxrdp - macOS 용 xrdp
## 개요
osxrdp 는 macOS에서 rdp 서버를 사용할 수 있게 해주는 xrdp의 비공식 모듈입니다.

## 기능
|기능|상태|
|------|---|
|부드러운 화면 제어 (H.264)|✅|
|로그온되지 않은 macOS 사용자를 사용한 제어|❌|
|세션 크기 조절 & 가상 해상도|❌|
|오디오|❌|
|클립보드|❌|
|파일 전송|❌|
|다중 모니터|❌|

## 제한 사항
* 원격으로 제어할 macOS 사용자는 반드시 '로그온' 한 상태여야 합니다. 이는 새 macOS 세션을 만들고 해당 세션에 사용자를 로그온할 방법을 찾지 못했기 때문입니다.
* osxrdp 는 아직 알파 버전 상태입니다. 오동작할 수 있으며, 상업적 용도로 사용하기 부적절합니다.

## 알려진 이슈
* 호스트 컴퓨터에 반드시 1개 이상의 물리 모니터가 연결되어 있어햐 합니다.
* 아직 가상 해상도를 지원하지 않으므로 호스트 컴퓨터의 모니터와 해상도가 다른 경우 품질이 저하될 수 있습니다.
* 마우스 커서의 모양이 상황에 맞게 변하지 않습니다.
* mstsc 를 사용하여 접속 시, 최초 접속 이후 mstsc 창을 최소화한 후 다시 열면 화질이 약간 떨어집니다. 이 현상이 왜 발생하는지는 아직 모르겠습니다.
  
## 지원 OS
macOS 12.4 이상\
Apple Silicon 및 Intel 맥을 지원합니다.
